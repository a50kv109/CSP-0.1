# CSP Roadmap

This document outlines the planned evolution of the **Cognitive Skill Passport (CSP)** protocol and ecosystem.

## Phase 1: Foundation (Current)
- [x] **CSP-0.1**: Initial protocol definition (DOI: 10.5281/zenodo.19639060).
- [x] **CSP-0.2**: Enhanced compatibility layer and passport structure.
- [x] **CSP-TRUST-CONCEPT**: Distributed trust and execution attestation layer proposal.

## Phase 2: Tooling (Upcoming)
- [ ] **CSP Inspector**: Local tool for passport validation and TDI/DBE analysis.
- [ ] **CSP CLI**: Command-line interface for skill generation and verification.
- [ ] **Passport Validator**: Official JSON/YAML schema for automated validation.

## Phase 3: Ecosystem (Future)
- [ ] **DBE Implementation**: Standardized Dynamic Benchmark Engine reference.
- [ ] **Skill Marketplace Protocol**: Decentralized infrastructure for skill exchange.
- [ ] **CSP 1.0 Stable**: Official stable release.
